### 6. 상세 학습 내용
ft_atoi.c 고려해야할 점.
1. 공백이 숫자, 부호 앞에 오는 경우 
"\n\t\n42" = 42
"     +42" = 42
2. 공백이 부호와 숫자 사이에 오는 경우
"- + - +42" = 42
3. 공백이 뒤에 오는 경우
"42\r\t\n\t\v\f42" = 42
"42 42" = 42
4. "1234asdf" = 1234
피신 기간에 학습하였던 atoi와는 또다른 느낌이다.

스터디 참여 : 백준 문제를 풀었다.
입출력 관련하여 풀었다. EOF에 대해서 배웠다.
